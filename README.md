# Linux Basics for DevOps

This repository contains my Linux learning for DevOps, including
basic commands, file system navigation, permissions, and practice tasks.

## Topics Covered
- File & directory commands
- Viewing files
- Searching logs
- Permissions basics

## Why Linux for DevOps?
Linux is the foundation of DevOps. Most servers run on Linux,
and DevOps engineers must be comfortable with terminal operations.

## Commands Practiced
- ls, cd, pwd
- mkdir, rm, cp, mv
- cat, less, head, tail
- grep

## Practice Work
I practiced these commands daily on terminal and documented
my learning and outputs.
